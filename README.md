# Portofolio Guru TKJ

Website portofolio sederhana berbasis HTML dan CSS yang dibuat sebagai project pembelajaran cloud computing untuk siswa SMK TKJ.

## Tujuan Project

- Mengenalkan dasar HTML dan CSS
- Mengenalkan penggunaan GitHub
- Mengenalkan konsep hosting cloud
- Melatih siswa deploy website menggunakan GitHub Pages

## Teknologi yang Digunakan

- HTML
- CSS
- GitHub
- GitHub Pages

## Struktur File

- index.html
- materi.html
- jobsheet.html
- kontak.html
- style.css
