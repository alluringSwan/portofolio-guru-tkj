Jobsheet 1: Deploy Website ke GitHub Pages

Tujuan: Siswa mampu membuat website sederhana dan menguploadnya ke cloud menggunakan GitHub Pages.

Langkah Kerja

  1. Membuat file index.html
  2. Membuat file style.css
  3. Membuat repository GitHub
  4. Mengupload file website
  5. Mengaktifkan GitHub Pages
  6. Menguji website melalui browser
