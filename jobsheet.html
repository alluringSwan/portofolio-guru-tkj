Jobsheet 1: Membuat Website Portofolio

Tujuan: Siswa mampu membuat website portofolio sederhana menggunakan HTML dan CSS.

Alat dan Bahan

  • Laptop/komputer
  • Browser
  • Akun GitHub
  • Koneksi internet

Langkah Kerja

  1. Buat file index.html
  2. Buat file style.css
  3. Tambahkan halaman materi, jobsheet, dan kontak
  4. Upload project ke GitHub
  5. Aktifkan GitHub Pages
  6. Uji website melalui browser

Jobsheet 2: Deploy Website ke GitHub Pages

Tujuan: Siswa mampu mempublikasikan website agar dapat diakses secara online.

Tugas Siswa

  • Membuat repository GitHub
  • Mengupload file website
  • Mengaktifkan GitHub Pages
  • Mengumpulkan link website
