Kumpulan project pembelajaran cloud computing dan web untuk siswa SMK TKJ

Beranda Materi Jobsheet Project Kontak

Daftar Project

Project 1: Website Portofolio

Membuat website portofolio sederhana menggunakan HTML dan CSS, lalu menguploadnya ke GitHub.

Project 2: Deploy ke GitHub Pages

Mempublikasikan website agar dapat diakses secara online menggunakan layanan GitHub Pages.

Project 3: Dokumentasi Praktik Cloud

Membuat dokumentasi hasil praktik siswa dalam bentuk website sebagai portofolio digital.

Tujuan Project

  • Melatih siswa memahami konsep website dan hosting.
  • Melatih siswa menggunakan GitHub sebagai tempat penyimpanan project.
  • Melatih siswa melakukan deploy website ke cloud.
  • Membangun portofolio digital siswa TKJ.

Output yang Dikumpulkan Siswa

  1. Link repository GitHub.
  2. Link website GitHub Pages.
  3. Screenshot tampilan website.
  4. Penjelasan fungsi file HTML dan CSS.
